# Mayank Vishwakarma — Personal Portfolio

A premium, interactive portfolio website built with **Next.js**, **Tailwind CSS v4**, and **Framer Motion**.

## ✨ Features

- 🎯 **Anti-Gravity UI** — Cursor-based magnetic hover, parallax, floating cards
- 🌊 **Particle Field** — Canvas-based neural network particle background
- 🎬 **Cinematic Animations** — Character-by-character text reveal, scroll-driven transitions
- 🃏 **3D Project Cards** — Perspective-tilting cards with glow effects
- 📊 **GitHub Stats** — Animated counters, contribution graph, language breakdown
- 🎨 **Glassmorphism** — Premium glass effects throughout
- 🌙 **Theme Toggle** — Dark/Light mode
- 📱 **Fully Responsive** — Optimized for all devices
- 🥚 **Easter Egg** — Konami code celebration effect
- ⚡ **Performance** — Lazy loading, code splitting, GPU acceleration

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build
```

## 🛠 Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Particles**: HTML5 Canvas
- **Fonts**: Inter, Outfit, JetBrains Mono (Google Fonts)
